# 딥시크 코드 (DeepSeek Code)

🇰🇷 **한국어 특화 AI 코딩 에이전트**

DeepSeek V3 기반으로 자연어 코딩 작업을 수행하는 CLI 도구입니다.
Claude Code처럼 파일 탐색, 코드 분석, 자동 수정을 지원합니다.

## 특징

- **한국어 최적화**: 한국어로 자연스럽게 코딩 작업 요청
- **파일 자동 탐색**: 프로젝트 구조를 분석하고 관련 파일 자동 탐색
- **코드 자동 수정**: 버그 수정, 기능 추가, 리팩토링 자동 수행
- **다양한 모드**: 단일 작업, 파이프라인, 멀티태스크, 인터랙티브
- **실시간 진행 표시**: 토큰 사용량 및 비용 실시간 확인

## 설치

```bash
npm install -g deepseek-code
# 또는
pnpm add -g deepseek-code
```

## 환경 설정

DeepSeek API 키가 필요합니다.

```bash
export DEEPSEEK_API_KEY="your-api-key"
```

API 키는 [DeepSeek Platform](https://platform.deepseek.com)에서 발급받을 수 있습니다.

## 사용법

### 기본 사용

```bash
# 단일 작업
dsc "로그인 기능 추가해줘"

# 버그 수정
dsc "이 에러 수정해줘: TypeError at line 42"

# 코드 리뷰
dsc "이 함수 리팩토링해줘"
```

### 파이프라인 모드

복잡한 작업을 분석 → 구현 → 검토 3단계로 처리합니다.

```bash
dsc "회원가입 기능 전체 구현해줘" --pipe
```

### 인터랙티브 모드

여러 작업을 비동기로 실행하며 대화형으로 작업합니다.

```bash
dsc -i
```

인터랙티브 모드 명령어:
- `<작업>` - 작업 입력 후 Enter → 백그라운드 실행
- `s` - 현재 실행 중인 작업 상태 확인
- `q` - 종료

### 멀티태스크 모드

여러 작업을 동시에 병렬 처리합니다.

```bash
dsc "a.py 만들어" "b.py 만들어" "c.py 만들어"
```

## 옵션

| 옵션 | 설명 |
|------|------|
| `-i, --interactive` | 인터랙티브 모드 |
| `-m, --model <id>` | 사용할 모델 선택 |
| `--pipe` | 파이프라인 모드 |
| `--multi` | 멀티태스크 모드 |
| `-l, --list` | 모델 목록 확인 |
| `-h, --help` | 도움말 |

## 지원 모델

- **DeepSeek V3.2** (기본) - 최신 모델, GPT-4 수준 성능
- **DeepSeek Coder** - 코딩 특화 모델

## 작동 방식

1. **프로젝트 분석**: list_directory로 프로젝트 구조 파악
2. **파일 탐색**: search_files로 관련 파일 검색
3. **내용 확인**: read_file로 파일 내용 분석
4. **코드 수정**: edit_file로 정확한 위치 수정
5. **실행/테스트**: bash로 빌드, 테스트 실행
